Les 4 premiers projets contenu dans ce repot exige de posséder:
- Soit les équipements électroniques que j'ai utilisé
- Soit allé sur le site tinkercard.com qui est un similateur qui permet d'utiliser des composants électronique, monter un circuit,
  éditer le code de la carte arduino et simuler le résultat.

 Pour ceux qui possèdent des équipements, j'ai personnellemnt utilisé l'éditeur ArduinoIDE pour le code et les équipements 
 sont des équipements de base : 
 - Des LED
 - carte ARDUINO (moi j'ai utiliser ARDUINO UNO mais ce n'est pas obligatoire, utiliser le modèle qui vous convient)
 - cables jumpers
 - Résistances
 - bouttons poussoir
 - BreadBoard

   Pour les utilisateurs de Tinkercard allez sur le site, ensuite sur la zone créer un nouvau circuit
   et suivez les étapes de réalisations ci-dessous.

 Pour réaliser les projets vous devez :
- Monter le circuit selon le schéma de la photo.
 - Ensuite copier coller le code dans votre éditeur (ou si vous voulais vous exercez écrivez le vous meme).
 - Enfin téléverser le code dans votre carte ARDUINO avec le cable(pas nécessaire sur tinkercard, cliquez sur simuler pour testez)
 - Et c'est bon votre projets électronique fonctionne.
